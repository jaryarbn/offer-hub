# Local Reference Snapshots

These files are local development snapshots of the Feishu documents used by the
Offer Hub backend. They are not synchronized automatically.

- `question-api.md`: confirmed question-module HTTP contracts and current
  implementation conventions.
- `data-dictionary.md`: confirmed MongoDB collection fields used by the question
  module.

Source priority for development:

1. Feishu interface documents define HTTP request and response contracts.
2. Feishu data dictionary defines persistence field names and types.
3. Feishu status/enumeration document defines enum values.

Snapshot date: 2026-07-21.
